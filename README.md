## ⚙️ 개발 환경

- Node.js / Express

# 📌 기능 기술
- Front-End :React 라이브러리를 이용해 프론트엔드 애플리케이션 개발
  - HTML, CSS(SCSS), JavsScript(ES6)
- Back-End : Node.js express 프레임워크와 MySQL 사용해 백엔드 애플리케이션 개발
- Front-End + Back-End
  - Frontend
    - Backend에 api 요청 작업
    - read, create, delete, update
  - Backend
    - CORS 문제 해결 : 요청하는 곳의 포트번호가 다르더라도 허용
